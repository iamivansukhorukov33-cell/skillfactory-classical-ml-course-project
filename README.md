# skillfactory-classical-ml-course-project
Course project on classical machine learning for predicting chemical compound properties
# Курсовая работа: классическое машинное обучение

## Описание проекта

В данной курсовой работе решается задача прогнозирования свойств химических соединений на основе числовых молекулярных дескрипторов.

Датасет содержит информацию о химических соединениях и целевых показателях:

- `IC50, mM`
- `CC50, mM`
- `SI`

Цель работы — провести разведочный анализ данных, построить модели регрессии и классификации, сравнить качество разных алгоритмов машинного обучения и сделать аналитические выводы.

## Структура проекта

- `01_eda.ipynb` — разведочный анализ данных
- `02_regression_ic50.ipynb` — регрессия для прогнозирования IC50
- `03_regression_cc50.ipynb` — регрессия для прогнозирования CC50
- `04_regression_si.ipynb` — регрессия для прогнозирования SI
- `05_classification_ic50_median.ipynb` — классификация IC50 > медианы
- `06_classification_cc50_median.ipynb` — классификация CC50 > медианы
- `07_classification_si_median.ipynb` — классификация SI > медианы
- `08_classification_si_more_8.ipynb` — классификация SI > 8
- `processed_chemical_data.csv` — обработанный датасет после EDA
- `Данные_для_курсовои_Классическое_МО.xlsx` — исходный датасет
- `report.docx` / `report.pdf` — аналитический отчёт

## Использованные методы

### Регрессия

- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

### Классификация

- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier

## Метрики качества

Для задач регрессии использовались:

- MAE
- RMSE
- R2

Для задач классификации использовались:

- Accuracy
- Precision
- Recall
- F1
- ROC-AUC

## Основные результаты

В задачах регрессии для `IC50, mM` и `CC50, mM` наиболее устойчивые результаты показала модель Random Forest.

Задача прогнозирования `SI` оказалась более сложной: качество регрессии было ниже, чем для IC50 и CC50.

В задачах классификации лучшие результаты показали модели Random Forest и Gradient Boosting. Особенно хорошее качество было получено для задачи `CC50 > медианы`.
